# Discord Announcement - OmenCore v3.6.2 (Stable)

OmenCore v3.6.2 is now live.

Release: https://github.com/theantipopau/omencore/releases/tag/v3.6.2

v3.6.2 is a stabilization release focused on reliability and consistency:
- fan/performance mode state now follows confirmed backend apply paths
- better RGB fallback behavior on unsupported hardware/effects
- expanded model/capability coverage for reported OMEN/Victus IDs
- Linux packaging and diagnostics improvements
- lower idle/tray/dashboard overhead and UI responsiveness cleanup

Downloads:
- OmenCoreSetup-3.6.2.exe (Windows installer)
- OmenCore-3.6.2-win-x64.zip (Windows portable)
- OmenCore-3.6.2-linux-x64.zip (Linux CLI + Avalonia GUI)

If you notice regressions, post model + BIOS version + a short repro in Discord and attach logs/diagnostics from OmenCore.
