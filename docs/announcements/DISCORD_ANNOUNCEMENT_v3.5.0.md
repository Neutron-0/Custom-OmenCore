🎉 OmenCore v3.5.0 is now live 🎉

This release focuses on reliability, clearer diagnostics, and safer tuning behavior.

Highlights:
- Clearer fan/profile wording so linked vs independent behavior is easier to understand
- New quick tuning diagnostics panel for screenshot-friendly CPU UV / GPU OC / conflict / startup-recovery status
- Better requested vs confirmed state handling across fan, tray, and tuning flows
- Tuning startup recovery guardrails and conflict detection improvements
- Fresh Windows installer, Windows portable, and Linux packages with published SHA256 hashes
